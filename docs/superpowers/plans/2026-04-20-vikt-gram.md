# Vikt i gram — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Byt ut kg-visning i hela UI:t mot gram, utan att ändra databasen.

**Architecture:** Konvertering sker enbart i presentationslagret. Databasen lagrar fortfarande `weight_kg`. En hjälpfunktion `formatWeightG(kg)` används för formaterad visning; registreringsformuläret dividerar med 1000 innan det sparar. Inga typer eller DB-kolumner ändras.

**Tech Stack:** Next.js (App Router), React, TypeScript, MUI, Supabase (oförändrad), Jest + React Testing Library
